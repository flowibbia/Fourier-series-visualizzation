Play with Sines

This project was inspired by the following article Purrier Series
The purpose of this project is just to play with sines, from the article linking above, you can see that you can approximate periodic function with sums of sines. A trivial sines is composed from an amplitude and a frequency, below you can see graphically how multiple sines are added togheter, and if you put right fourier coefficient for a function, you can approximate it. From the button rectangle wave and triangle wave, you get 15 coefficient for the relative function.

How to play

The amplitude is expressed in pixel, if the coefficient is too small, scale all coefficient multiplying for the same factor.
The n box is for the n-th coefficient and is a multiple of the fundamental frequency, don't worry for the value of this, it is irrelevant for the shape of the function.
